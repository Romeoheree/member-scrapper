## • API ID and API HASH Setup
* Go to [telegram](http://my.telegram.org)  and log in or use [@s4h_api_idbot](https://t.me/s4h_api_idbot) 

* Click on API development tools and fill the required fields.

* put app name you want & select other in platform Example :

* copy "api_id" & "api_hash" after clicking create app ( will be used in setup.py )


## • How To Install and Use In Termux

`$ pkg up -y`

`$ pkg install python -y`

`$ pkg install git`

`$ git clone https://github.com/evilgodtermux/member-scraper`

`$ cd member-scraper`

* Install requierments & Setup Configuration File. ( apiID, apiHash )

`$ python setup.py`

* To scrape members from group.

`$ python scraper.py`

* Add Scraped members to your group. 

`$ python adder.py`
